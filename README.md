## Stack

- [Nest](https://github.com/nestjs/nest)
- [MongoDB](https://mongo.org)

## Instalación

```bash
$ npm install
```

## Ejecución de la app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
